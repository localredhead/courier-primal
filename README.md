courier-primal
==============

Courier Prime with a slash in the zero.

Any and all licenses transfer from the original, found here: http://quoteunquoteapps.com/courierprime/

RoboFont was used to edit the original files to add a slash to the zero's.  All credit goes to the original author of the font.

RoboFont can be fount here: http://robofont.com/ and can be used to edit the .ufo files and generate the .TTF's/.OTF's.
